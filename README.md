# 👋 Hi there! I'm Jorge

I'm a master's graduate in Data Science & Business Analytics, currently working as a Data Scientist at Cognodata Consulting.

Although I've always been interested in Computer Science and related stuff, I really started being attracted by Data Science back in 2022, with a DIY home server project.
Through an awesome (and challenging) journey, I discovered what drives me. That was when I decided to study a master's degree.

Now, I really like to code and I focus on building predictive models and simplifying complex workflows.
